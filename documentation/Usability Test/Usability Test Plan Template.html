Usability Test Plan Section

Test Results for: Visia Chrome Extension

Success Criteria

  • 75% of users understand how to use the extension
  • 75% of users consider extension to be useful
  • 50% of users will use again the extension

Summary

Major findings:

  • 3 users intuitivelly understood how to use the extension
  • 3 users consider that corespondence between button icons and the actions that where triggered was wisely chosen
  • 3 users appreciate the minimal design and the fact that extension is easy to use
  • 4 users consider that extension can be improved by adding new features and more control upon actions

Post-test Questions Answers

The questions are subjective and open. Their purpose is to identify what can be done to improve the user experience.

Test Observations

Obs 1: For some users it was difficult to navigate to options page.

Obs 2: Extension is easy to use for the average person that had some interaction with a web browser before.

Obs 3: Design can be improved by making it to look better without adding extra complexity in it.

Potential Design Improvements

Main window(popup) can be made to remain on page without collapsing back.

Main window can be made to be draggable.

A shortcut to options page can be added in main window
